# poc-kafka
POC Kafka producer/consumer

This project is an example of a Kafka producer/consumer.

## How to run

````
docker-compose build
docker-compose up
````

## Evidences

Once running, you can view Kafka stats and messages via Control Center UI: http://localhost:9021/
